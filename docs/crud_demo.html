CRUD HTTP Demo (C + SQLite)

Minimal REST API running alongside the Tenge project to demonstrate web/DB integration and performance-friendly design.

GET
/todos

List todos.

POST
/todos

Create todo: {"title":"...","priority":1}

GET
/todos/{{id}}

Get one by id.

PATCH
/todos/{{id}}

Update fields: {"done":true}

DELETE
/todos/{{id}}

Delete by id.

How to run locally

  1. Run server (from ../tenge): 
    make -C ../tenge examples/crud_todos http-run
# server listens on http://127.0.0.1:8080
  2. Smoke test: 
    ./scripts/crud_smoketest.sh   # requires curl & jq

curl snippets

# list
curl -s http://127.0.0.1:8080/todos | jq .

# create
curl -s -X POST http://127.0.0.1:8080/todos \
  -H "Content-Type: application/json" \
  -d '{"title":"Write blog post","priority":2}' | jq .

# get
curl -s http://127.0.0.1:8080/todos/1 | jq .

# patch
curl -s -X PATCH http://127.0.0.1:8080/todos/1 \
  -H "Content-Type: application/json" \
  -d '{"done":true}' | jq .

# delete
curl -s -X DELETE http://127.0.0.1:8080/todos/1 | jq .

SQLite: ../tenge/examples/crud_todos/todos.db. Recreate table: make -C ../tenge examples/crud_todos purge.
